# Sample Videos Directory

For testing the playback functionality, place test video files here.

Supported formats:
- .mp4
- .mkv
- .avi
- .mov
- .wmv
- .webm

Example test approach:
1. Download a small public domain video (e.g., from archive.org)
2. Place it in this directory
3. Update MainWindow.xaml.cs to auto-load it for testing
